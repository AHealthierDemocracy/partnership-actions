# Vot-ER Partnership Actions

This repository tracks partner outreach, event toolkits, and communications resources for Vot-ER's partnership program.

It is organized by year and event, with reusable templates available in `/templates`.

## Structure

```
partnership-actions/
├── 2026/
│   └── events/
│       └── april-7-know-your-rights/
│           ├── event-details.md
│           ├── email-template.md
│           ├── sms-template.md
│           └── compliance-notes.md
└── templates/
    ├── email-template.md
    └── sms-template.md
```
